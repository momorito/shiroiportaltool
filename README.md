# Androidアプリ「白井市ポータル」

## このアプリは？
Android上で動作する[白井市](http://www.city.shiroi.chiba.jp/toppage.html)のホームページを閲覧したりする機能を持つアプリケーションです。

## 主な機能
### ホームページ閲覧機能

### 新着情報閲覧機能

### ナッシー号検索機能

### 電話機能

### 各種SNSアクセス機能

## ダウンロード
[Google Playからダウンロード](https://play.google.com/store/apps/details?id=jp.tanikinaapps.shiroiportaltools)
